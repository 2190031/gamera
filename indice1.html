Funcionamiento General del Sistema


El sistema está diseñado de forma que funcione muy parecido al trabajo de una oficina cualquiera, esto es a través de listados de elementos que pueden ser:

  • Datos de Bancos
  • Datos de Clientes
  • Datos de Préstamos
  • Datos de Cuotas
  • Datos de Recibos de Pagos

Por lo tanto se tiene una ventana para el manejo de cada listado. En función del volumen de datos a manejar se usará una ventana de listado simple, o una ventana con filtro de datos. Como se había mencionado anteriormente, el sistema funciona a través de la web, es por esto que se debe limitar la cantidad de información a transmitir en cada petición al servidor porque puede congestionar el tráfico de la misma y por ende la interfaz de usuario visualizar el error de datos fuera de tiempo (Timeout en inglés).z
